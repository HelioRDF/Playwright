## Pré-requisitos

- Instalar o Node

- Dependências
```npm init -y ``` ou ```npm install ```
```npm install playwright``` 
```npm install expect```

- Instalar Playwright teste
```npm i -D @playwright/test ```

- Executar arquivo .js
```node teste.js```

- Executar arquivo .spec
```npx playwright install```
```npx playwright test```
```npx playwright test --ui```
